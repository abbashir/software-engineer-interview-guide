# Visual Studio Code Markdown Preview Guide

Visual Studio Code includes built-in support for previewing Markdown (`.md`) files.

## Open Markdown Preview

1. Open a Markdown (`.md`) file in VS Code.
2. Use one of the following shortcuts:

### Windows / Linux

- **Open Preview:** `Ctrl + Shift + V`
- **Open Preview to the Side:** `Ctrl + K`, then `V`

### macOS

- **Open Preview:** `Cmd + Shift + V`
- **Open Preview to the Side:** `Cmd + K`, then `V`

## Alternative Methods

- Right-click anywhere in the Markdown editor and select **Open Preview**.
- Click the **Open Preview** (📄🔍) icon in the top-right corner of the editor.
- Open the **Command Palette** (`Ctrl + Shift + P` / `Cmd + Shift + P`) and search for:
  - `Markdown: Open Preview`
  - `Markdown: Open Preview to the Side`

> **Tip:** Using **Open Preview to the Side** lets you edit the Markdown on one side while seeing the rendered output update live on the other.
